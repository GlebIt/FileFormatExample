Пример изображения в формате .hui

Ниже приведен интерактивные пример нашего формата. В таблице приведены значения каждого байта файла

Первые два байта 0 3 размер 3 пикселя по вертикали, далее 0 пустой байт,
далее два байта 0 2 размер 2 пикселя по горизонтали, далее 0 пустой байт. Далее идут байты хрянаящие значения RGB для каждого пикселя.
Значения RGB каналов можно изменять чтобы увидеть как это меняет изображение. Ссылка на пост: Создадим свой формат файла для изображений?
